Класс ImgHelper

Класс ImgHelper позволяет пропорционально уменьшать картинку, приводя ее к заданным размерам без искажения.
Для этого вычисляются пропорциональные размеры и по ним происходит уменьшение (увеличение) исходной картинки  по меньшей стороне, 
большая сторона обрезается по краям. Если исходная картинка пропорциональна, то обрезки не происходит.
В результате качественая картинка в десятки МБ весит десятки либо пару сотен Кб в хорошем качестве.

Mysql.sql

Содержит различные запросы к БД на обновление таблиц, добавление функции и триггеров, связанных с email рассылкой, расчетом реитингов
и созданием и наполнением таблицы поиска из различных таблиц сайта.
